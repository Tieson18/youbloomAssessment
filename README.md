# Youbloom React Assessment

## Overview
This is a React application created for the Youbloom Frontend Developer Internship assessment.  
The app includes a **login page**, **main user list page**, and **detail page** with search functionality.  
It demonstrates use of **React**, **React Router**, **MUI components**, **Tailwind CSS** (optional), and **international phone number validation**.

---

## Features
- Login with any valid international phone number (E.164 format)
- Display a list of users fetched from a public API
- Search users by name, email, or username
- User detail view with company and address info
- Mobile-responsive layout
- Clean navigation between pages

---

## Tech Stack
- React
- React Router
- Material-UI (MUI)
- Tailwind CSS (optional)
- Jest + React Testing Library for unit testing
- Vite as development server

---

## Setup Instructions

1. Clone the repository: git clone <REPO_URL>

2. Install dependencies: npm install

3. Run tests: npm test

4. Run the development server: npm run dev
