# Youbloom React Assessment

## Overview
This is a React-based assessment project for Youbloom. The application demonstrates a **login workflow**, **user listing**, **search**, and **user detail pages**, using modern React best practices. The project is responsive, validates international phone numbers, and integrates a mock authentication context.

**Key features:**
- Login page with international phone validation (E.164 format)
- Main page displaying users fetched from API
- Search functionality for filtering users
- Detail page for individual users
- Responsive design with MUI
- Mock authentication and routing
- Unit tests for login validation using Jest & React Testing Library

---

## Technologies Used
- React (Functional Components + Hooks)  
- React Router (Routing & Navigation)  
- Material-UI (UI Components)  
- Tailwind CSS (Optional, styling)  
- Jest + React Testing Library (Unit Tests)  
- Axios (API requests)  
- Vite (Build Tool)
